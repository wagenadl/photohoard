PHOTOHOARD(1)
=============
:doctype: manpage

NAME
----
photohoard - Photography collection management

SYNOPSIS
--------
*photohoard*

DESCRIPTION
-----------
The *photohoard*(1) command starts the PHOTOHOARD Photography
Collection Manager.

OPTIONS
-------
None.

BUGS
----
There may be bugs. Please contact the author if one bites you.
